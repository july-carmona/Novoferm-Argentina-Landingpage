Descripción del producto

Sistema compacto enrollable para huecos con tránsito intensivo

Novoroll Industrial
Persiana enrollable rápida para uso intensivo

El Novoroll es la solución enrollable de Novoferm diseñada para instalarse en huecos interiores con tránsito intensivo. Su velocidad de maniobra de ~1 m/seg la hace ideal para evitar corrientes entre sectores gracias a la gran estanqueidad que proporcionan las guías laterales con cepillos en ambas caras. Con clasificación EI(1) 60 y EW 180, ofrece protección cortafuego certificada.

  • Lona de tejido de poliéster con revestimiento PVC (900 gr/m²)
  • Estructura autoportante de acero galvanizado
  • Motor trifásico con sistema paracaídas integrado y encoder
  • Fotocélula, banda de seguridad y control programable IP54 de serie
  • Guías laterales de acero galvanizado con cepillos para máxima estanqueidad
  • Amplitud de colores RAL disponibles para adaptarse a cada proyecto

Aplicaciones

¿Dónde se instala el Novoroll?

Especialmente indicado para instalaciones industriales con tránsito frecuente y necesidad de estanqueidad.

Fábricas
Separación de sectores productivos con alta circulación de carretillas y operarios.
Almacenes
Compartimentación en centros de distribución con tránsito intensivo de montacargas.
Industria alimentaria
Separación de zonas de temperatura controlada y áreas de procesamiento.
Farmacéutica
Control de accesos entre áreas limpias y zonas de producción controlada.
